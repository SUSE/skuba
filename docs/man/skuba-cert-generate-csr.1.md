# NAME
generate-csr - generates in-cluster services CSR and key

# SYNOPSIS
**images**
[**--help**|**-h**]
*images*

# DESCRIPTION
**generate-csr** generates in-cluster services CSR and key which is used for custom CA or trusted CA to sign the certificate

# OPTIONS

**--help, -h**
  Print usage statement.
