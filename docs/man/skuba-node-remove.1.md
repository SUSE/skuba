# NAME
remove - remove a node to a state prior to join or bootstrap

# SYNOPSIS
**remove**
[**--help**|**-h**]
*remove* *node-name*

# DESCRIPTION
**remove** will remove a node from the cluster via kubernetes

# OPTIONS

**--help, -h**
  Print usage statement.
